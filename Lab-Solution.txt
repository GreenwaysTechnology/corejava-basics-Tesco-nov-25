‚úÖ Step 2: Create the ATM Class
ATM.java
public class ATM {
    private int pin = 1234;
    private double balance = 5000.00;

    public void verifyPin(int enteredPin) throws InvalidPinException {
        if (enteredPin != pin) {
            throw new InvalidPinException("‚ùå Invalid PIN entered!");
        }
    }

    public double getBalance() {
        return balance;
    }

    public void withdraw(double amount) {
        if (amount <= 0) {
            throw new IllegalArgumentException("‚ùå Amount must be greater than zero");
        }
        if (amount > balance) {
            throw new InsufficientBalanceException("‚ùå Insufficient balance!");
        }
        balance -= amount;
        System.out.println("‚úÖ Withdrawal successful. New balance: " + balance);
    }
}

import java.util.Scanner;

public class ATMLab {

    public static void main(String[] args) {

        ATM atm = new ATM();

        try {
            System.out.print("Enter PIN: ");
            int pin = sc.nextInt();

            atm.verifyPin(pin);

            System.out.println("‚úÖ Login successful!");
            System.out.println("Current balance: " + atm.getBalance());

            System.out.print("Enter amount to withdraw: ");
            double amount = sc.nextDouble();

            atm.withdraw(amount);

        } catch (InvalidPinException e) {
            System.out.println(e.getMessage());

        } catch (InsufficientBalanceException e) {
            System.out.println(e.getMessage());

        } catch (Exception e) {
            System.out.println("‚ùå System error: " + e.getMessage());

        } finally {
            System.out.println("üîí Session ended.");
            sc.close();
        }
    }
}

        System.out.println("üìß Email sent: " + message);
    }
}

public class SMSNotification implements NotificationService {
    public void send(String message) {
        System.out.println("üì± SMS sent: " + message);
    }
}

public class WhatsAppNotification implements NotificationService {
    public void send(String message) {
        System.out.println("üí¨ WhatsApp message: " + message);
    }
}

‚úÖ Step 3: Factory Class (Real-World Pattern)
public class NotificationFactory {

    public static NotificationService getService(String type) {
        if(type.equalsIgnoreCase("email")) {
            return new EmailNotification();
        } else if(type.equalsIgnoreCase("sms")) {
            return new SMSNotification();
        } else if(type.equalsIgnoreCase("whatsapp")) {
            return new WhatsAppNotification();
        } else {
            throw new IllegalArgumentException("Invalid notification type");
        }
    }
}

Member Inner Class ‚Äì Bank System
‚úÖ Scenario

Bank account where inner class handles ATM security.

‚úÖ Code
public class BankAccount {

    private double balance = 10000;

    // ‚úÖ Member Inner Class
    class Security {
        void verifyUser() {
            System.out.println("üîê User verified successfully");
        }
    }

    public void withdraw(double amount) {
        Security security = new Security();
        security.verifyUser();

        if (amount <= balance) {
            balance -= amount;
            System.out.println("‚úÖ Withdrawal successful: " + amount);
        } else {
            System.out.println("‚ùå Insufficient balance");
        }
    }

    public static void main(String[] args) {
        BankAccount acc = new BankAccount();
        acc.withdraw(2000);
    }
}

Real-Time Case Study ‚Äì Order Processing System
‚úÖ Scenario

Different inner classes handle payment, shipping and invoice.

public class OrderService {

    public void processOrder() {

        // ‚úÖ Local Inner Class
        class Payment {
            void pay() {
                System.out.println("üí≥ Payment processed");
            }
        }

        // ‚úÖ Member Inner Class
        class Shipping {
            void ship() {
                System.out.println("üöö Order shipped");
            }
        }

        Payment payment = new Payment();
        Shipping shipping = new Shipping();

        payment.pay();
        shipping.ship();
    }

    public static void main(String[] args) {
        new OrderService().processOrder();
    }
}
